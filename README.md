#### API INTEGRATION(WEATHER,CRYPTO,NEWS)
## Overview
This project demonstrates integration of multiple APIs to fetch real-time data from news,weather and cryptocurrencyservices.
It showcases hoe to work with both authenticated APIs(requiring API keys) and public APIs(no key required).

## Objectives
* Understand API integration concepts.
* Fetch real-time data from multiple APIs
* Parse JSON responses
* Display structured and readable output
* Implement error handling

## Technologies used
* Python
* Requests library
* JSON

## APIs Used
# Weather API
* wttr.in
* Provides real-time weather information
* No API key required
# Crypto API
* CoinGecko
* Provides cryptocurrency prices
* No API key required
# News API
* NewsData.io
* Provides real-time news headlines
* Requires API key

## API Requirement
* wttr.in - No API key required
* CoinGecko - No API key required
* NewsData.io - Requires API key

* Get weather updates
